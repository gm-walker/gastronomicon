# gastronomicon

A flavor-pairing app that'll allow you to find pairings amongst your ingredients.

# running the app

I have used **Docker** to simulate how this application would function. Don't fret; you can run the app without it.

## prerequisites

Running this _kind of_ requires Docker to function... so you _might_ want to get that installed.

## steps

```bash
# start the development environment
docker compose up

# close the development environmnet
docker compose down

# see just the app running (doesn't require docker; no backend tho)
npm run app
```
